# Sales-Analysis
Retail sales analysis project in Python. Computes KPIs (total revenue, average order value, top regions), analyzes seasonality, and visualizes top products and trends with attractive charts.

# Sales Analysis Project

This project analyzes retail sales data to compute KPIs and visualize trends.

## Features
- Compute KPIs: Total Revenue, Average Order Value, Top Regions, Top Products
- Seasonality analysis (monthly revenue trends)
- Attractive visualizations saved as PNG files
- One-page summary with recommendations

Install Requirements:-
pip install pandas matplotlib seaborn

Run the script:-
python sales analysis.py

Output:-
. monthly_trend.png

. top_products.png

. top_regions.png
